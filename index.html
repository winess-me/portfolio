My projects

Issue tracker web app

The application is a simple issue tracker, in which a user can sign in and then add, edit and delete issues, as well as change their status and sort them. Try it out yourself at the link below!

The user's data is stored cloud-based MongoDB storage. The app uses Node.JS, Express with Mongoose and Typescript on Backend; Vue CLI, Vue Router, Vuex, Typescript and Axios on Frontend.

CSS code is styled according to BEM methodology. Project file structure uses MVC design pattern. Authorisation is done through Auth0 3rd-party service using JWT. The app is deployed on Heroku.
